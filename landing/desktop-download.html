How to Install

Takes less than 60 seconds

1

Download the Installer

Choose the version for your operating system above and download the installer file.

Windows: Use the .exe installer or the portable .zip.
macOS: Open the .dmg and drag the app into Applications.
Linux: Run the AppImage or install the .deb with sudo dpkg -i.
2

Run the App

Launch AnswerSnap Desktop from your applications menu or desktop shortcut.

The first launch may prompt you to allow accessibility/screen-recording permissions. This is required for the app to capture the screen around your double-click.
3

Double-Click to Answer

Open any quiz, PDF, or form. Double-click the question text to get an instant answer.

Ctrl+Shift+H shows/hides the answer window.
Ctrl+Shift+C copies the last answer to your clipboard.
Ctrl+Shift+Q quits the app.
