How to Install

Takes less than 60 seconds

1

Unzip the File

After downloading, extract the ZIP file to a folder on your computer. Remember where you saved it.

Windows: Right-click → Extract All
Mac: Double-click the ZIP
2

Open Chrome Extensions

Open Chrome and go to the extensions page. Enable Developer mode in the top-right corner.

chrome://extensions
Toggle the "Developer mode" switch in the top-right corner of the page.
3

Load Unpacked

Click "Load unpacked" in the top-left, then select the folder you extracted in Step 1.

You should see the Cheatly icon appear in your Chrome toolbar. You're all set!
