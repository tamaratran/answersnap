How to Install

Takes less than 30 seconds

1

Add to Chrome

Click the button above to open the Chrome Web Store. Then click "Add to Chrome" and confirm by clicking "Add extension".

Make sure you're using Google Chrome or a Chromium-based browser (Edge, Brave, etc.).
2

Pin the Extension

Click the puzzle piece icon in Chrome's toolbar, then click the pin icon next to Cheatly for easy access.

Pinning keeps the Cheatly icon visible in your toolbar so you can see when it's active.
3

Start Answering

Navigate to any quiz or assignment. Double-click a question and Cheatly will instantly find and select the answer.

Cheatly works on Canvas, Google Forms, Quizlet, and any other website.

Start Using Cheatly

Two ways to get instant answers

Double-Click

Double-click any question text on the page. The answer appears instantly.

Alt + A

Toggle the extension on/off at any time. When on, every double-click triggers an answer.
